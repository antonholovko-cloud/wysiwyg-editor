HTML Insertion Feature

Insert custom HTML directly into the editor. Try adding tables, custom styled divs, or any HTML content!

  1. Place your cursor where you want to insert HTML
  2. Click the HTML button (</>) in the toolbar
  3. Enter your HTML code in the dialog
  4. Click "Insert" to add it to the editor

Sample HTML to Try:

<table style="width: 100%; border-collapse: collapse;">
  <tr style="background-color: #f0f0f0;">
    <th style="border: 1px solid #ddd; padding: 8px;">Name</th>
    <th style="border: 1px solid #ddd; padding: 8px;">Value</th>
  </tr>
  <tr>
    <td style="border: 1px solid #ddd; padding: 8px;">Item 1</td>
    <td style="border: 1px solid #ddd; padding: 8px;">100</td>
  </tr>
</table>
